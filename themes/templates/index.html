  • 安装环境

    安装 Node.js

    安装 cnpm: npm install -g cnpm --registry=https://registry.npm.taobao.org

  • 初始化脚手架

    导航到你的开发目录后 git clone git@github.com:m-develop/boilerplate.git

  • 简单配置

    将目录名改为你的项目名，并将 package.json 中的 name 改为你的项目名

  • 开发调试

    运行 cnpm install，安装需要的依赖

    完成安装后，运行 cnpm start ，开启本地服务，通过浏览器访问 http://127.0.0.1:8001
